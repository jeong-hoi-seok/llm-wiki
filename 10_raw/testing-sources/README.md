## 처리 기준

- 테스트 개념과 실무 판단 기준만 `wiki/테스트/`로 추출한다.
- 원문별 맥락은 `wiki/원문요약/`에 별도 요약으로 남긴다.
- 특정 회사의 도구 선택은 그대로 규칙화하지 않고, 재사용 가능한 판단 기준으로 변환한다.
