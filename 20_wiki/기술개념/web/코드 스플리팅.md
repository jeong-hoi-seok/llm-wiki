# 코드 스플리팅

## 한 줄 요약
큰 JS 파일을 작은 파일로 나누고 **필요한 시점에 필요한 코드만 로드**하는 최적화. [[번들링]] 기법.

## 핵심
- 모든 페이지 코드를 한 번에 로드하면 안 쓰는 리소스까지 받아 성능↓.
- 방문한 페이지에 필요한 코드만 **동적 로드** → 초기 로딩↑, 리소스 낭비↓.

## 방법
- **`import()`** — 모듈을 필요 시점에 비동기 로드 (정적 `import` 대신).
  ```js
  // 한 번에 로드 → 필요 시 로드
  const mod = await import("./module.js");
  ```
- 라우트 단위 분할(페이지별), Lazy Loading(React `lazy`/`Suspense`)과 결합.
