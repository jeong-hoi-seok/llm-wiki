# CLAUDE.md

Claude 전용 진입 파일.

상세 운영 규칙은 `AGENTS.md`가 기준이다. 작업 전 반드시 `AGENTS.md`를 먼저 읽고 따른다.

요약:
- 답변과 문서는 한국어로 작성한다.
- `10_raw/`는 원본 보관소이므로 수정하지 않는다.
- `20_wiki/`만 생성·수정한다.
- 질문 답변 전 `20_wiki/index.md`를 먼저 읽는다.
- 작업 후 `20_wiki/index.md`와 해당 날짜의 `20_wiki/log/YYYY-MM-DD.md`를 갱신한다.
- Obsidian wiki는 공통 프론트엔드 지식 베이스이며, 특정 프로젝트 규칙은 프로젝트 `docs/`가 우선한다.
